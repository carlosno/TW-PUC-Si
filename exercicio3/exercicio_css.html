Instruções

  1. A entrega dos exercicios e individual.
  2. cada exercicio deve ser implementado em um arquivo separado. se o arquivo não for especificado no
    enunciado, use o padrão exercício_NUMERO.html, como, por exemplo,exercicio_1.html
  3. Crie uma pasta de nome css comun a todos os exercicios onde devem ser mantidos os arquivos (css) externos
    nescessários.

  • Todos os arquivos HTML e CSS produzidos devem ser validados (http://html5.validator.nu/ e
    http://www.ceeportal.com/css-validator/) e formatados (http://www.freeformater.com/html-formatter.html e
    http://ccsbeautify.com/) antes de serem entregues.
  • Não use espaços, letras maiúsculas ou símbolos nos nomes dos arquivos.
